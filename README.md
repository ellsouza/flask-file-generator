# Flask File Generator

Aplicação simples em Python com Flask para gerar e baixar um arquivo `.txt` via interface web.

## Funcionalidade

- Geração automática de arquivo `.txt`
- Download direto no navegador
- Interface web básica

## Tecnologias

- Python
- Flask
- HTML / CSS
